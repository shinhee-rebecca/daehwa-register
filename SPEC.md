# 대화상점 인명부

## 앱 설명

대화상점이라는 독서모임의 참여자 정보를 저장, 관리하기 위한 프로젝트



## 사용자 구분 및 권한

### 운영자

저장하는 데이터
- 성별 / 이름 / 전화번호

권한 및 기능
- 모든 참여자 정보 조회하기 / 수정하기 / 삭제하기
- 리더 추가하기 / 수정하기 / 삭제하기

### 리더

저장하는 데이터
- 성별 / 이름 / 전화번호 / 담당모임

권한 및 기능
- 자신이 담당하고 있는 모임의 참여자 정보 조회하기

### 참여자

저장하는 데이터
- 성별 / 나이 / 이름 / 개월수 / 첫등록월 / 전화번호 / 회비 / 재등록 / 최등록 / 현재모임 / 비고 / 과거 등록 모임 리스트



## 앱 기능

### 참여자 관련

- 성별 / 나이 / 이름 / 개월수 / 첫등록월 / 전화번호 / 회비 / 재등록 / 최등록 / 현재모임 등으로 참여자 정보를 검색하여 조회하기 (이때 한 페이지에 15명씩 표 형식으로 보여주며 페이징 기능을 지원, 현재 페이지의 참여자를 엑셀로 다운받기, 현재 조회된 모든 참여자를 엑셀로 다운받기 등을 지원)
